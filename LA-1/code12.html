Emergency Helpline Numbers in India

1. National Emergency Number
Icon / Image 🚨
Description A single, pan-India integrated emergency number providing immediate assistance from Police, Fire, and Health departments.
Helpline Number 112
Official Email ID contact@112.gov.in


2. Police Control Room
Icon / Image 🚓
Description For reporting crimes, seeking immediate police intervention, and assistance in matters of public safety.
Helpline Number 100
Official Email ID controlroom@police.gov.in


3. Fire Brigade
Icon / Image 🚒
Description Immediate response line for reporting fires, rescues, and other hazardous material emergencies.
Helpline Number 101
Official Email ID emergency@fire.gov.in


4. Medical Emergency / Ambulance
Icon / Image 🚑
Description Dedicated service for dispatching ambulances and providing critical pre-hospital medical care for life-threatening situations.
Helpline Number 102 / 108
Official Email ID medical@health.gov.in


5. Women Helpline
Icon / Image 👩🏽‍🦰
Description Provides 24-hour emergency response, counseling, and assistance to women facing violence or seeking refuge.
Helpline Number 1091
Official Email ID complaint@ncw.nic.in


6. Child Helpline
Icon / Image 🧸
Description A 24-hour free emergency phone service for children in need of care and protection.
Helpline Number 1098
Official Email ID dial1098@childlineindia.org.in


7. Senior Citizen Helpline
Icon / Image 🧓
Description Provides assistance, emotional support, and rescue services for elderly individuals facing abuse or medical emergencies.
Helpline Number 14567
Official Email ID contact@dosje.gov.in


8. Disaster Management Services
Icon / Image 🌪️
Description National hotline for reporting and seeking aid during natural calamities, earthquakes, and large-scale disasters.
Helpline Number 1078
Official Email ID controlroom@ndma.gov.in


9. Cyber Crime Helpline
Icon / Image 💻
Description A dedicated hotline to report online financial fraud, cyber stalking, and other internet-related crimes immediately.
Helpline Number 1930
Official Email ID complaint@cybercrime.gov.in


10. Road Accident Emergency Service
Icon / Image 💥
Description Immediate contact channel for highway emergencies, road accidents, and requesting trauma care units on highways.
Helpline Number 1073
Official Email ID highways@morth.gov.in
