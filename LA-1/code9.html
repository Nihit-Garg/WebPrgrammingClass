Adding Images to a Web Page

This tutorial explains the three ways to source and display images on your website using the <img> tag.

1. From an External Website Link (URL)

You can use an absolute URL pointing to an image hosted on another server. While convenient, you rely on the external server's uptime for your image to load.

2. From an Image Stored on the Local Computer

While developing locally, you can reference an absolute file path on your own computer's hard drive. However, this image will break when the site is published because other users cannot access your local file system.

3. From an Image Placed Inside the Project Folder

This is the standard and most reliable method. By placing an image inside your website's project directory, you use a relative path. This ensures the image moves with the HTML file when deployed to a server.
